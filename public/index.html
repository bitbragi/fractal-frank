What is Frank?

Frank is a Model Context Protocol (MCP) server that gives AI assistants deep knowledge of Fractal Bitcoin development.

Quick Start

Get Frank running in under a minute

Terminal 
# Clone and setup
git clone https://github.com/bitbragi/frank-mcp.git
cd frank-mcp
python -m venv venv
source venv/bin/activate
pip install fastmcp httpx python-dotenv

# Add to Claude Code
claude mcp add frank -- ~/frank-mcp/venv/bin/python ~/frank-mcp/frank_mcp.py
